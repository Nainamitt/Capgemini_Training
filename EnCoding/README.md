In Machine Learning, Encoding means converting categorical (text) data into numerical format so that algorithms can understand and process it.

Most ML algorithms work only with numbers, not text like Male, Female, Red, Blue, etc.
So encoding is used to transform these categories into numbers.


Types of Encoding

1️⃣ Label Encoding
When categories have no problem with numeric labels

2️⃣ One Hot Encoding
When categories are nominal (no order)

3️⃣ Ordinal Encoding
When categories have order
