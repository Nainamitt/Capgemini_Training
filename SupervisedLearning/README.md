Supervised Learning is a type of Machine Learning where the model is trained using labeled data.
This means the dataset already contains the input (features) and the correct output (label/target), and the model learns the relationship between them.


Types of Supervised Learning

1. Classification
Used when the output is categorical.

Common Algorithms:

Logistic Regression , 
K-Nearest Neighbors (KNN) , 
Decision Tree


2. Regression
Used when the output is continuous numeric value.

Common Algorithms:

Linear Regression , 
Multi-linear Regression
